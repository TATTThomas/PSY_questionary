Goal setting


From the interviews or biography of some top athletes, we could know setting goals as soon as possible is a key point to induce efficiency and successful rate.

Here’s an example of Major League’s player Shohei Ohtani during high school, he wrote all his goals in a baseball’s strike zone, which the ultimate target will be written on the center of the zone.
Goal setting can bring several benefits to players
Reference(https://sportspsyc.wordpress.com/2020/06/16/goal-setting-for-athletes/)：

Setting goals effectively

To set a goal and conduct it wisely, we advise using some strategy as assistant like "SMART"
1. S-Specific: Specific targets.
2. M-Measurable: Digitize targets.
3. A- Attainable: Set the target clearly, well-difination.
4. R-Relevant: All small targets must be high relevant to the main target.
5. T-Time-Bound: Set time limits for every target.

Notice:

Regular self inspection is needed for enduring target achieving and avoid wrong deviations.
