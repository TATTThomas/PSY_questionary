基本管理頁

  1. 創建新測驗者
    皆需輸入標頭,為任意英文字母字串,會冠在個人ID前。
    創建多位新測驗者功能不會回傳創建的帳號密碼,需自行於下載完整表格的帳號密碼下載所有帳密,挑選具有對應標頭的帳號密碼複製。
  2. 下載個人資料
    於目標ID填寫想搜尋的ID即可

認知作業

  1. 更新表格
    請先下載原資料,根據原資料進行修改,存回時須為*.csv副檔名,此外,須為純csv格式,若更新表格後發現相關功能失效,請檢查表格是否為標準CSV,利用notepad++開啟,檢查是否為純逗號,換行符組成,不可含',' '\r\n' 以外符號用來區分格式
  2. 下載
    會在最下方選擇變換時進行下載,接收資料後會挑出下載連結,請直接點選。若想重新下載,任意切換各選項即可。

環境 名稱

  1. 安裝node
    https://nodejs.org/en/
    下載windows安裝檔即可(*.msi)
  2. 安裝mongoDB
    參考:https://ithelp.ithome.com.tw/articles/10244454
  3. 安裝全域 pm2
    使用npm安裝
    npm install pm2 -g
  4. 安裝git
    clone 本系統
  5. pm2 發布以及windows開機自啟
    參考:https://ithelp.ithome.com.tw/articles/10243636
  6. 開通外部防火牆
    參考:https://ithelp.ithome.com.tw/articles/10253979
    發布=>PQ_main at port:1337
    發布=>PQ_MDB at port:1338
    發布=>PQ_SRS at port:1339
  7. 開通各電腦本地防火牆
    參考:
    1. 控制台\系統及安全性\Windows Defender 防火牆
    2. 進階設定
    3. 輸入規則
    4. 新增規則
    5. 連接阜->填入上一步的port
  8. DB填入初始lock

  • lock

    • users
      • name:usersCount
      • count:1

    cmd指令

mongo
use lock
db.users.insert({"name":"usersCount","count":1})
  1. 環境變數設置
    • PQ_main資料夾找到.env檔案
      輸入:
      SRSurl=http://140.116.98.155:1339/
      (填入參數為影片伺服器網址)
    • PQ_MDB資料夾找到.env檔案
      輸入:
      ID=admin
      password=0000
      (填入參數為管理者帳密 可自訂)
  2. 系統重啟
    參考第5步,利用windows service重啟服務即可
